Player Search

Research, Learn, Report

We allow you to search through any player's pitches and at-bats with quick facets on things like batted ball angles, pitch speed, pitch types, hit type, and many others.

The Data

Using the atbat-mongodb project, we pull pitch and at-bat data from each day's baseball games and store it in our own MongoDB database.

Data from each day will be available by 6AM EST on the following day.

Got Issues or Suggestions?

If you find any issues with this site or simply want to make a suggestion, you can create an issue on the GitHub project page.

Be sure to search first to make sure your issue doesn't already exist. If it does, feel free to comment on the existing issue. I'm more likely to prioritize an issue with lots and lots of comments.
